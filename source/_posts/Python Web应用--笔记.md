**Python web开发介绍**
打开本书,说明你对 Python web开发是有兴趣的,或者正打算开始学习使用 Python做web
开发,又或者已经是一个web开发者。那么,用Python做Web开发的人员需要具备哪些技
术能力呢?

- 至少熟悉一种 Python web框架。
- 熟悉 Python语法。
- 熟悉数据库、缓存、消息队列等技术的使用场景、使用方法等。
- 日常能使用 Linux或Mac系统工作。
- 有性能调优经验,能快速定位问题。
- 对HTML/CSS/ javascript有一定了解,有使用经验。
